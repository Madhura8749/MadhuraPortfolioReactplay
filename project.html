During my 5th semester of engineering, I have done one miniproject based on Database management system. And the project tittle is ,"Server Notification Management system". While doing this project, I have learnt how to manage Time, and how to communicate and present project. Server Notification Management System is an application or set of application that lets network engineers to manage networks independent component inside a bigger network. It is used to manage server issues. Notification will pop-up when the server is not reachable.

In this project, I have used MYSQL as backend, and for frontend - Java Swings. The platform used is JDBC jar.
